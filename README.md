#### 交易系统撮合引擎
  一款使用golang编写的，采用优先级队列实现的订单排序、订单撮合、输出委托深度等的开源程序。

#### 功能列表
  - [x] 输出委托深度
  - [x] 订单撮合  
  - [x] 支持限价单
  - [x] 取消订单
  - [x] 统一数字小数位
  - [ ] 支持市价单


#### 委托深度http接口数据预览
```
{
  "ask": [
    ["1.0001", "19960"],
    ["1.0000", "10"]
  ],
  "bid": [
    ["0.9999", "20"],
    ["0.9998", "20"]
  ]
}
```
["1.0001", "19960"] => [价格，数量]
